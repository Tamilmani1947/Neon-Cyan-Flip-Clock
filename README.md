# Neon Cyan Flip Clock

A modern, futuristic digital flip clock built using **HTML**, **Tailwind CSS**, and **Vanilla JavaScript**. This clock features a sleek neon aesthetic, 3D flip animations, and real-time time & date display.

## Features

- **Futuristic Neon Aesthetic**  
  Dark background with neon cyan and blue glow effects, housed in a glassmorphism-inspired container.

- **3D Flip Animation**  
  Each digit (hours, minutes, seconds) flips realistically when it changes using CSS 3D transformations.

- **Orbitron Typography**  
  Bold, geometric font to enhance the sci-fi digital look.

- **Dynamic AM/PM Glow**  
  - AM: Red Glow  
  - PM: Neon Cyan Glow

- **Fully Responsive**  
  Looks great on all devices, from mobile phones to large desktop screens.

- **Real-Time Display**  
  Shows current time (12-hour format with AM/PM) and full date (DD/MM/YYYY).
